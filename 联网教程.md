## 为什么外地朋友连不上？

```
你的电脑 ←→ 家里的路由器 ←→ 互联网 ←→ 朋友的路由器 ←→ 朋友的电脑
     ↑              ↑
  内网IP        没有公网IP
  (192.168.x.x)  (NAT转换)
```

**核心问题：** 你家的路由器没有独立的公网IP，外面的人无法直接找到你的电脑。

## 方案一：局域网对战（最简单）

✅ **适合：** 在一起的朋友，同一个WiFi/路由器下

### 操作步骤

1. **主机操作（创建房间的人）：**
   - 右键点击 `启动服务器.bat` → 以管理员身份运行
   - 按 `Win+R`，输入 `cmd`，回车
   - 输入 `ipconfig`，找到 IPv4 地址（例如：`192.168.1.100`）

2. **朋友操作（加入房间的人）：**
   - 确保连接同一个WiFi
   - 浏览器访问 `http://主机IP:8765/online.html`
   - 例如：`http://192.168.1.100:8765/online.html`
   - 点击「连接」按钮

3. **开始游戏：**
   - 主机点击「创建房间」
   - 朋友输入房间码点击「加入房间」

### 常见问题

| 问题 | 解决方法 |
|------|---------|
| 网页打不开 | 检查是否同一WiFi，主机防火墙是否放行 |
| 服务器连接失败 | 主机是否运行了启动服务器.bat |
| 加入房间失败 | 房间码是否正确，双方是否连接同一服务器 |

## 方案二：云服务器对战（推荐）

✅ **适合：** 异地朋友对战，稳定可靠
💰 **费用：** 约20-50元/月（轻量应用服务器）

### 推荐服务器

| 服务商 | 配置 | 价格 | 推荐度 |
|--------|------|------|--------|
| 腾讯云轻量 | 2核2G 4M | ~30元/月 | ⭐⭐⭐⭐⭐ |
| 阿里云轻量 | 2核2G 5M | ~35元/月 | ⭐⭐⭐⭐ |
| 华为云 | 2核2G | ~25元/月 | ⭐⭐⭐ |

### 部署步骤

1. **购买云服务器：**
   - 操作系统选 **Windows Server** 或 **Linux**（推荐Linux更省资源）
   - 地域选离你们最近的
   - 带宽选4M以上就够了

2. **开放端口：**
   - 登录云服务器控制台
   - 找到「安全组」/「防火墙」设置
   - 添加入站规则：TCP 8765 端口放行

3. **上传文件到服务器：**
   - 把 `server.py` 传到服务器
   - 安装 Python 和 aiohttp：
     ```bash
     pip install aiohttp
     ```

4. **启动服务器：**
   ```bash
   python server.py
   ```
   看到「服务器启动在 http://0.0.0.0:8765」就成功了

5. **玩家连接：**
   - 浏览器访问 `http://服务器公网IP:8765/online.html`
   - 双方都连接同一个服务器
   - 一人创建房间，一人加入

## 方案三：内网穿透（免费）

✅ **适合：** 不想花钱，偶尔玩一玩
⚠️ **缺点：** 速度可能不稳定，有流量限制

### 推荐工具

| 工具 | 免费额度 | 难度 | 推荐度 |
|------|---------|------|--------|
| 花生壳 | 1M带宽/1G流量/天 | 简单 | ⭐⭐⭐⭐ |
| ngrok | 1M带宽 | 中等 | ⭐⭐⭐ |
| cpolar | 1M带宽 | 简单 | ⭐⭐⭐ |

### 花生壳使用步骤

1. 下载安装花生壳客户端
2. 注册账号登录
3. 添加映射：
   - 应用名称：像素机甲对战
   - 内网主机：127.0.0.1
   - 内网端口：8765
   - 外网端口：自动分配
4. 启动映射，得到一个外网地址（例如：`xxx.xicp.net:12345`）
5. 朋友访问 `http://xxx.xicp.net:12345/online.html`

## 方案四：虚拟局域网（ZeroTier）

✅ **适合：** 多个人经常一起玩，免费
⚠️ **缺点：** 双方都要安装软件

### 使用步骤

1. 去 zerotier.com 注册账号
2. 创建一个网络（Network），得到 Network ID
3. 双方都下载安装 ZeroTier 客户端
4. 双方都加入同一个 Network ID
5. 在 ZeroTier 后台勾选允许两台设备
6. 双方都能看到分配的虚拟IP（例如：`10.147.x.x`）
7. 一方运行 `启动服务器.bat`
8. 另一方访问 `http://对方虚拟IP:8765/online.html`

## 常见问题排查

### 🔴 网页完全打不开

**检查清单：**
- [ ] 服务器是否启动？
- [ ] 地址和端口是否正确？
- [ ] 是否在同一网络？
- [ ] 防火墙是否放行？

**测试方法：**
- 主机自己先测试 `http://localhost:8765` 能不能打开
- 主机用内网IP测试能不能打开
- 逐步排除问题

### 🟡 网页能打开但服务器连接失败

**检查清单：**
- [ ] 服务器地址输入是否正确？
- [ ] 末尾有没有加端口 `:8765`？
- [ ] 服务器是否在运行？

**提示：** 游戏页面右上角会显示连接状态

### 🟢 能连接但游戏卡顿

**解决方法：**
- 尽量靠近路由器
- 关闭占用带宽的软件（下载、视频等）
- 云服务器选更近的地域
- 降低游戏画质（这个游戏像素风本来就不卡）

## 📞 还是连不上？

按这个顺序检查：

1. **本地测试：** 你自己电脑能不能玩？
2. **局域网测试：** 同一WiFi下的手机能不能访问？
3. **端口检查：** 用站长工具的「端口扫描」检测8765端口是否开放
4. **防火墙：** 系统防火墙 + 路由器防火墙 + 云服务器安全组，都要检查
